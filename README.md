# #compraaospequenos
> Covid-19 crisis: help local shops survive and thrive

## Local development

Copy `.env.example` to `.env` and add configuration keys (ask @pvhee).

To run the project locally

```bash
yarn install
yarn develop
```

To create a production build into the `\.public` folder

```bash
yarn build
```
